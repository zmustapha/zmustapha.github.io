Laws of Propositional Logic
De Morgan’s laws:
¬( p ∧ q ) ≡ ¬p ∨ ¬q
¬( p ∨ q ) ≡ ¬p ∧ ¬q
Idempotent laws:
p ∨ p ≡ p
p ∧ p ≡ p
Associativity laws:
( p ∧ q ) ∧ r ≡ p ∧ ( q ∧ r )
( p ∨ q ) ∨ r ≡ p ∨ ( q ∨ r )
Commutativity laws:
p ∧ q ≡ q ∧ p
p ∨ q ≡ q ∨ p
Distributive laws:
p ∧ ( q ∨ r ) ≡ ( p ∧ q ) ∨ ( p ∧ r )
p ∨ ( q ∧ r ) ≡ ( p ∨ q ) ∧ ( p ∨ r )
p ∨ ( q ∨ r ) ≡ ( p ∨ q ) ∨ ( p ∨ r )
p ∧ ( q ∧ r ) ≡ ( p ∧ q ) ∧ ( p ∧ r )
Identity laws:
p ∧ T ≡ p
p ∨ F ≡ p
Domination laws:
p ∨ T ≡ T
p ∧ F ≡ F
Double negation law:
¬¬p ≡ p
Absorption laws:
p ∨ (p ∧ q) ≡ p
p ∧ (p ∨ q) ≡ p
Implication to disjunction law:
p → q ≡ ¬p ∨ q
IFF to implication law:
p ↔ q ≡ ( p → q ) ∧ ( q → p )
